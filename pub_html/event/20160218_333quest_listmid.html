3·3·3 퀘스트 이벤트

새로운 모습으로 찾아온 EBSMath가 재미있는 퀘스트 이벤트를 진행합니다! 매일 EBSMath에서 퀘스트를 완료하면 경품 당첨의 행운이!

퀘스트 선물
  • 매일 출석 + 퀘스트 100% 달성 인원 중 추첨 00명 : 세미 3D 피규어 + 보조배터리
  • 퀘스트 100% 달성 인원 중 추첨 00명 : 보조배터리
  • 기간 : 3월 7일(월) ~ 3월 27일(일)
  • 발표 : 3월 31일(목)

  1. 1. 참여하기 - 이벤트 참여하기 버튼을 클릭
  2. 2. 일일퀘스트 - 매일 새로운 일일 퀘스트를 완료
  3. 3. 스탬프 - 퀘스트를 완료하고 스탬프 모으기
  4. 4. 상품증정 - 상품 당첨 기준까지 스탬프를 모으면 추첨을 통해 상품 증정!
