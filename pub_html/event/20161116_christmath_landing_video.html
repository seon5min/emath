메리 크리스MATH 이벤트

EBSMath는? 동영상, 웹툰, 게임 등의 수학 콘텐츠를 학생이 스스로 재미있게 배울 수 있도록 도와주는 EBS의 자기주도 수학 학습 사이트입니다.

  • 이벤트 기간 : 2016년 12월 7일 (수) ~ 2017년 1월 5일 (목)
  • 당첨자 발표 : 2017년 1월 9일 (월)

페이스북 트위터 카카오스토리

상품은 추!첨!을 통해 제공되며, 도장을 많이 모을수록 당첨확률이 높아집니다.

  1. 1등 : 아이패드 Pro 1명
  2. 2등 : 세미 피규어 40명
  3. 3등 : 치킨 + 콜라 세트 50명
  4. 4등 : 주말 영화관람권 60명
  5. 참가상 : 치즈케익 100명
