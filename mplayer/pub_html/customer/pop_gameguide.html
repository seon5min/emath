Autograph Player 설치

  • 1. Autograph Player 설치
    노출되어 있는 설치 이미지를 통하여 Autograph Player를 설치합니다.

    1. 1-1. 브라우저 상단의 [알림 표시줄]을 클릭하여 Autograph Player 설치 시작

    2. 1-2. [설치] 버튼 클릭

    3. 1-3. 설치 완료

Cabri 3D Plug-in 설치

  • 1. 사용 환경 확인 : 인터렉티브 학습창을 사용하기 위한 사용자 시스템 환경을 확인합니다.

    1. Windows

      · Windows 98 (Internet Explorer 5 이상), Me, NT4, XP, Vista

    2. Mac OS

      · Mac OS X, version 10.3 또는 그 이상

    3. PC 최소사양

      · 800MHz 또는 그 이상의 CPU, 256MB 또는 그 이상의 RAM, OpenGL 호환되는 메모리 64M 또는
        그 이상의 그래픽 카드

  • 2. Cabri 3D Plug-in 설치
    노출되어 있는 설치 이미지를 통하여 Cabri 3D Plug-in을 설치합니다.

    1. 2-1. [Download Cabri 3D Plug-in] 클릭을 통하여 설치 시작

    2. 2-2. 페이지 하단에 [실행] 버튼 클릭

    3. 2-3. [실행] 버튼 클릭

    4. 2-4. 파일 실행 중

    5. 2-5. [실행] 버튼 클릭

    6. 2-6. 설치 언어 선택 후 [확인] 버튼 클릭

    7. 2-7. [다음] 버튼 클릭

    8. 2-8. '사용자 계약에 동의합니다.' 항목 선택 후 [다음] 버튼 클릭

    9. 2-9. 설치 진행 중

    10. 2-10. 설치 완료 후 [완료] 버튼 클릭
