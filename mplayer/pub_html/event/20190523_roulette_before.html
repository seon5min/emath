    이벤트 응모 전 확인 바랍니다.
  • 이벤트 당첨자의 경우 경품 수령 정보 입력 기간(당첨자 발표일로부터 10일) 내에 배송 정보를 입력해야 합니다.

  • 개인 정보 이용에 대한 동의를 하지 않은 참여자는 당첨자 선정에서 제외됩니다.
  • 이벤트에 당첨된 고객님의 개인 정보는 경품 배송을 위해 제휴업체인 젤라블루에 취급 위탁하며, 사용 후 폐기됩니다.
  • 타인의 개인정보를 도용하거나 부정한 방법으로 이벤트 참여 시 당첨에서 제외, 혹은 당첨 취소될 수 있습니다.
  • 실물 경품과 모바일 경품 모두 당첨자 경품 수령 정보 입력 기간 종료 이후 일괄 발송됩니다.
